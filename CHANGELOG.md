# Changelog

All notable changes to the Second Last Clipboard extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial Raycast extension development
- Quick paste command for second last clipboard item
- Visual clipboard history viewer
- Comprehensive error handling
- TypeScript implementation with strict typing
- ESLint and Prettier configuration

### Features
- **Quick Paste Command**: Instant no-view command for pasting second last clipboard item
- **History Viewer**: Interactive interface showing up to 6 recent clipboard items
- **Search Functionality**: Filter clipboard history by content
- **HUD Confirmations**: Visual feedback showing pasted content
- **Action Panel**: Copy, paste, and refresh actions in history viewer
- **Keyboard Shortcuts**: Full keyboard shortcut support
- **Error Handling**: Graceful handling of empty clipboard history

### Technical
- React + TypeScript architecture
- Raycast API v1.101.1 integration
- Modern React Hooks implementation
- Comprehensive error boundaries
- Production-ready build configuration
